Frigid Launcher

Casual Mobile Game

Project Overview

Frigid Launcher is a casual mobile game developed by a small group of friends. The game features wave-based gameplay where players control a "launcher" to shoot polar bears appearing behind obstacles. Players earn points by hitting more bears, which can be exchanged for new launchers in the shop.

I contributed as a developer on this project, focusing on gameplay mechanics, UI implementation, and systems for enemies, the shop, encrypting save data, and customization. This project, which we developed post-university in about a month, allowed me to gain valuable experience in areas I hadn't explored much during my studies, such as UI development and encryption.

Key Features

  • UI and Customisation System Implementation
  • Encrypted Data Saving System
  • Wave-Based Gameplay
