Immersify

A cutting-edge dental and nursing education application

Immersify is an innovative educational platform that leverages XR technology to enhance dental and nursing education. The application provides interactive, immersive simulations for students to practice procedures in a virtual environment.

Key Features

  • Interactive XR-based learning modules
  • Real-time feedback and assessment tools
  • Multiplayer collaboration for group learning
  • Compatibility with VR headsets and mobile devices

Technologies Used

  • Unity 3D
  • C#
  • XR Toolkit
  • Oculus & HTC Vive support
